按钮

轻松创建拥有不同样式的漂亮按钮。

用法

在<a>元素或<button>元素中添加.uk-button类即可使用按钮组件。在<button>元素中添加disabled属性可以禁用按钮。

示例

Link

Code

<a class="uk-button" href="">...</a>
<button class="uk-button" type="button">...</button>
<button class="uk-button" type="button" disabled>...</button>

注意 如果在较小的视窗中试图一行展示多个按钮，可以给它们添加一个顶部外边距（top margin）。再在它们的父元素中添加 效果组件中的data-uk-margin属性即可。

色彩修饰

这里有几个可以修饰颜色的类。添加下列类中的一个，即可得到不同的外观。

示例 类 描述
.uk-button-primary 着重强调在一系列按钮中的重要性。
.uk-button-success 表示成功或积极的行为。
.uk-button-danger 表示危险或消极的行为。
Link .uk-button-link 保留按钮行为的同时，使其看起来更像一个连接。

尺寸修饰

可以在一个按钮中添加.uk-button-mini, .uk-button-small或.uk-button-large 类使该按钮更大或更小。

满宽按钮

添加网格组件中的.uk-width-1-1类， 该按钮将占满整个容器宽度。

示例

Code

<button class="uk-button uk-width-1-1 uk-margin-small-bottom">...</button>
<button class="uk-button uk-width-1-1">...</button>

按钮组

创建一个按钮组，在包裹着按钮们的<div>元素中添加.uk-button-group类即可。

示例

Link
Link
Link
Link

Code

<div class="uk-button-group">
    <a class="uk-button" href="">...</a>
    <button class="uk-button">...</button>
    <button class="uk-button">...</button>
</div>

JavaScript

你可以添加data-uk-button属性来实现切换按钮状态。

你可以通过JavaScript切换按钮状态。只需要添加data-uk-button属性即可。

示例

Code

<button class="uk-button uk-button-primary" type="button" data-uk-button>Button</button>

复选按钮

像复选框一样切换一组按钮的状态，需要将这些按钮包裹在带有data-uk-button-checkbox属性的<div>元素中。这属性也能用于按钮组。

示例

Code

<div data-uk-button-checkbox>
    <button class="uk-button">...</button>
    <button class="uk-button">...</button>
    <button class="uk-button">...</button>
</div>

单选按钮

想要按钮组像下面例子中一样只能按下一个？用带有data-uk-button-radio属性的div元素包裹一组按钮就行。这个也能用于按钮组。

示例

Code

<div data-uk-button-radio>
    <button class="uk-button">...</button>
    <button class="uk-button">...</button>
    <button class="uk-button">...</button>
</div>

带下拉菜单的按钮组

使用按钮组将按钮分成左边是标准行为的按钮和右边是带下拉菜单的按钮，仅需将他们用一个<div>元素包裹，然后在<div>中添加data-uk-dropdown="{mode:'click'}"属性即可。 当然，下拉菜单也可以用于按钮组中的一个按钮。
